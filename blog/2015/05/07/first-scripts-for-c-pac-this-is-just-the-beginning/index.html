First Scripts for C-PAC: This Is Just the Beginning

May 7th, 2015 3:40 pm

Hi everyone, this is the presentation for my blog. Here, I will write about my experience with C-PAC development. Our intention here is to develop a series of scripts that allow us (C-PAC users) to calculate some measures over non-linear fMRI timeseries and integrate them within C-PAC. Our aim is to extend and improve C-PAC’s features.

In this post I will try to explain what I’ve done in my first two scripts for C-PAC. So, basically, this is my first approach to C-PAC; I need still to learn a lot (this is the main objective of the program) about C-PAC’s structure, workflow etc. . The goal of this first scripts is to say ‘hi’, have some feedback from main developers and take a first hands-on with the code.

So, I was taking a brief look to C-PAC’s repo on on GitHub and to C-PAC’s forums and I saw this and though it could be a good exercise to try to code it (and afterwards, I also made the calculation of GC). What these 2 scripts make is to calculate correlations and GC between ROI-time series extracted from an fMRI file (this is done using a ROI-mask with the gen_roi_timeserier.py function which is already in C-PAC).

My code is in the copy I forked, in a new branch called series_mod. I saved my scripts in a folder of the same name.
